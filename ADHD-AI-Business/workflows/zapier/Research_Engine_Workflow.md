# Zapier Workflow — Research Engine

## Goal
Run ADHD research skills and save results to a content database.

## Trigger Options
- Manual ChatGPT/Zapier MCP request
- Daily scheduled trigger
- Weekly planning trigger

## Steps
1. Run Find Trending ADHD Topics.
2. Run Parent Question Finder.
3. Run Medical Research Monitor.
4. Save top topics to Notion, Airtable, or Google Sheets.
5. Generate a daily research brief.
6. Send the brief to the user for review.

## Human Approval
Required before turning research into publishable content.
